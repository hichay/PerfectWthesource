Stolen Vehicle
by Scotty1944
https://www.facebook.com/Scotty1944/

Feature
- ขโมยรถตามจุดที่เราตั้งค่าใน Config
- สามารถตั้งให้รถคันที่ปล่อยให้ขโมยได้ มาในเวลาที่กำหนดในแต่ละวัน
- กำหนดจุดส่งรถได้ (สามารถทำสุ่มได้)
- ผู้เล่นที่เป็นตำรวจสามารถนำรถกลับคืนมาสถานีตำรวจได้
- ระบบรางวัลที่ใช้แบบ Battle X สามารถให้เป็นเงิน, เงินแดง, อาวุธ และสิ่งของ (สามารถสุ่มแบบกาชาก็ได้นะ :3)
- ระบบจัดการ Vehicle Network เพื่อที่ทั้งเซิฟจะได้เห็นแบบเดียวกัน
- ระบบ Discord Webhook แจ้งว่ามีใครขโมยจนไปส่งสำเร็จบ้าง
- รองรับ One Sync Infinity ในการแสดงรถบนแผนที่

* AI ตำรวจจะใช้ได้เฉพาะเซิฟที่ไม่ได้ปิด NPC และทำตามคู่มือเท่านั้น (Infinity ใช้ไม่ได้จ้าเพราะ NPC หาย)
- AI ตำรวจ จะไล่ล่าคุณถ้าคุณขโมยรถคันที่ตั้งค่าไว้
- เปิดปิด AI ตำรวจได้ตามใจชอบ
- สามารถปรับให้หนี AI ตำรวจก่อนจะไปส่งได้

**วิธีติดตั้ง**
1. โยนลง resources ของ fivem
2. เพิ่ม start scotty-stolen-vehicle ลงใน server.cfg
3. ใส่ license key ที่ได้จากผู้ขายใส่ลงในไฟล์ config_sv.lua
4. ปรับแต่งของที่สามารถทำได้ในไฟล์ config.lua
